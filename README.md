# 🍽️ Saini Chat Point

A responsive restaurant website built with **Flask** showcasing menu items, customer reviews, and contact information for Saini Chat Point.

## Features

* Hero section with restaurant highlights
* Menu with prices and Best Seller tag
* Customer reviews and ratings
* Contact details and timings
* Smooth scroll animations
* Mobile-friendly design

## Tech Stack

* Python
* Flask
* HTML
* CSS
* JavaScript

## Run Locally

```bash
pip install -r requirements.txt
python app.py
```

Open: `http://127.0.0.1:5000`
